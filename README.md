# Sistema de Gestão de Supermercado - LP2

Projeto desenvolvido em Java utilizando conceitos de Programação Orientada a Objetos:

- Herança
- Polimorfismo
- Abstração
- Encapsulamento
- Composição

## Funcionalidades
- Cadastro de produtos
- Cálculo de preço final com regras específicas
- Carrinho de compras
